General

Domain Keywords High weight · 2 pts each

Key entities for your domain — company names, brand names, organisations. One per line.

Sector Keywords Lower weight · 1.5 pts each

Sector nouns that characterise your domain. One per line.

Deal Activity Keywords Primary · 2.5 pts

Verbs and nouns indicating deal activity (acquisitions, investments, etc.). One per line.

Deal Signal Keywords Secondary · 1 pt

Supporting deal vocabulary. One per line.

Deal Type Labels

Map keywords to category labels for grouping. Format: keyword = Label. One per line.

Source Tier 2 Score: 7/10

Mainstream business/industry press. One per line.

Source Tier 3 Score: 5/10

Specialist trade press. One per line.

Blocklist Hard-blocked

Sources to block entirely (rumour sites, spam, unverified blogs). One per line.
